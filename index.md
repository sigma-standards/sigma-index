# SIGMA — Global Standards Index

**Free. Machine-readable. Human-navigable.**

Search and understand global standards across health, climate, trade, governance, technology, and beyond — without subscriptions.

## Current Status

**Phase:** Foundation build + generated release artifacts (May 2026)

**Current generated bundle:** 88,187 entries, 20,130 relationship edges, and coverage across all 40 canonical domains.

**Current staged candidates:** UN/OHCHR core human-rights treaty records and WHO IRIS normative metadata candidates are staged for curator review before release promotion.

## Core Goals

- Build a unified index of cross-jurisdictional standards and frameworks.
- Keep data machine-readable (CSV/JSON/Parquet) and human-friendly.
- Maintain open governance with transparent version history.

## Published Site

The GitHub Pages site is generated by `scripts/build_static_site.py`. It renders navigation, release downloads, documentation links, canonical domain coverage, source registry status, public search, and the current release metrics from the same `dist/` files used for downloadable data.

To preview the published structure locally:

```bash
make release
python3 scripts/build_static_site.py
```

To build the same Pagefind search bundle used by GitHub Pages:

```bash
make pagefind-search
```

The search surface includes `public/search.html`, Pagefind-readable generated record pages, and `public/search-index.json` as a local fallback.

## Google Sheet Sync

The shared `000_SIGMA_MASTER_DATABASE` Google Sheet can be synchronized into the processed data layer with `make sync-google-sheet`. The importer preserves the 22-field release schema and ignores temporary curation metadata until those fields are formally promoted.
