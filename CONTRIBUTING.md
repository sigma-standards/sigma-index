# Contributing to SIGMA — Unified Global Standards Index

Thank you for your interest in contributing to SIGMA! This project aims to build the world's most complete public index of global standards, treaties, frameworks, guidelines, and classification systems.

## Ways to Contribute

1. **Add New Entries**: Submit new standards, bodies, or treaties via GitHub Issues or Pull Requests.
2. **Correct Errors**: Report and fix inaccuracies in existing data.
3. **Expand Domains**: Help research and add entries for underrepresented domains.
4. **Improve Documentation**: Enhance README, guides, or schema documentation.
5. **Code Contributions**: Develop scripts for data processing, validation, or automation.

## Getting Started

1. Fork the repository on GitHub.
2. Clone your fork locally.
3. Create a virtual environment: `python -m venv venv && source venv/bin/activate`
4. Install dependencies: `pip install -e .`
5. Make your changes.
6. Run tests if available.
7. Submit a Pull Request.

## Data Contribution Guidelines

- All entries must include source URLs.
- Follow the SIGMA schema (see SCHEMA.md).
- Ensure data is accurate and verifiable.
- Use SIGMA ID format for new entries.

## Code Style

- Use Python 3.8+
- Follow PEP 8.
- Add docstrings to functions.
- Write tests for new functionality.
